# 🍕 SpyPizza – Sales Management System

SpyPizza is a web-based food ordering and sales management system built using **Python**, **Django**, **HTML**, and **CSS**. It allows users to browse and order items like pizza, burgers, sandwiches, cold drinks, and more.

## 🚀 Features

* 🍕 Order Pizza, Burgers, Sandwiches, Cold Drinks, etc.
* 🛒 Add to Cart functionality
* 📦 Order Management System
* 🧾 Admin Panel for managing products and orders
* 🔍 Dynamic product display
* 📱 Responsive UI (HTML & CSS)
* ⚡ Fast backend using Django

---

## 🛠️ Tech Stack

* **Backend:** Python, Django
* **Frontend:** HTML, CSS
* **Database:** SQLite (default Django DB)
* **Server:** XAMPP (for local environment support)

## 📂 Project Structure

```
SpyPizza/
```

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/spypizza.git
cd spypizza
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3️⃣ Install Dependencies

```bash
pip install django
```

### 4️⃣ Run Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

### 5️⃣ Start Server

```bash
python manage.py runserver
```

### 6️⃣ Open in Browser

```
http://127.0.0.1:8000/
```

---

## 🧑‍💻 Admin Panel

To access admin panel:

```bash
python manage.py createsuperuser
```

Then visit:

```
http://127.0.0.1:8000/admin/
```
